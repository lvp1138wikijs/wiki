Test if Time Zone Enabled or Not

To see if the tables are already loaded, enter mysql and type:

SELECT CONVERT_TZ('2013-03-17 12:00:00', 'GMT', 'America/New_York');

If the response looks like:

+--------------------------------------------------------------+ | CONVERT_TZ('2012-03-17 12:00:00', 'GMT', 'America/New_York') | +--------------------------------------------------------------+ | 2013-03-17 08:00:00                                          | +--------------------------------------------------------------+

That means the tables are loaded already and no further action is required.

If the response looks like:

+--------------------------------------------------------------+ | CONVERT_TZ('2012-03-17 12:00:00', 'GMT', 'America/New_York') | +--------------------------------------------------------------+ | NULL                                                         | +--------------------------------------------------------------+

That means the table must be loaded in MySQL.

Load Time Zone Table in MySQL

In linux the time zone information is in /usr/share/zoneinfo. So type the following command and hit enter

mysql_tzinfo_to_sql /usr/share/zoneinfo | mysql -u root -p mysql
Password:

Then type the mysql root password. 

Restart MySQL, and repeat the initial test.
