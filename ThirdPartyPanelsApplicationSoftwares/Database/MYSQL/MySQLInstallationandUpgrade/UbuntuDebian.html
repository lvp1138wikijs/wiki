MySQL Installation On Ubuntu/Debian

To install MySQL, run the following command from a terminal prompt of ubuntu.

apt-get install mysql-server

During the installation process you will be asked to enter a password for the MySQL root user. Put a password for mysql root user.

Once the installation is complete, the MySQL server should be started automatically. You can run the following command from a terminal prompt to check whether the MySQL server is running:

netstat -tap | grep mysql

When you run this command, you should see the following line or something similar:

tcp 0 0 localhost:mysql *:* LISTEN xxxxx/mysqld


If the server is not running correctly, you can type the following command to start it:

/etc/init.d/mysql restart
