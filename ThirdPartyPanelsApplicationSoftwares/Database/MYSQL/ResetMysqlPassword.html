How to Reset Mysql password 

Login to your server.

Next stop the mysql server.

# /etc/init.d/mysql stop

Next start the mysql server with skip grant table options.

# mysqld_safe --skip-grant-tables# mysql -u rootmysql> use mysql;

Next we change the current password using the below command:

mysql> update user set Password=PASSWORD('password_new') where user='root';

 Here we replace password_new with our new password.

mysql> flush privileges;

mysql> exit

Restart the mysql daemon.

service mysql restart
