# FTP software

File transfer Protocol (ftp) is a protocol used to transfer files over the web.

 

ftp file transfer can be done through command line as well as through free ftp software.

 

Some of the ftp software are:

Filezilla
Cute FTP
FTP Explorer
Elite FTP
