Now we can submit account reset, suspend / reactivate account, cancel account from the new admin portal. 

Always submit cancellation only through the client portal.

To do that follow the steps

Submit Reset

  1. Search for a shared account from Global Search in https://secure.serverpoint.com/cgi-bin/cp/engine/index.cgi using the username
  2. You will see the account details in the search results.
  3. Click on the Edit button
  4. Change the value of Status to  Send Reset. Then click on the Tick Mark button.

Suspend Account

  1. The first 3 steps are same as the above.
  2. Change the value of Status to Temporary suspension and then click on the Tick Mark button. 

System will automatically suspend the account within next few minutes.

Reactivate Suspended Shared Account

  1. The first 3 steps are same as the first procedure.
  2. Change the value of Status to Activate and then click on the Tick Mark button. 

System will automatically activate the account within next few minutes.

Cancel Shared Account - (Do not use this procedure. Always submit cancellation through the client portal)

  1. The first 3 steps are same as the first procedure.
  2. Change the value of Status to Cancel and deletion and then click on the Tick Mark button.

System will automatically cancel and delete the account within next few minutes.
