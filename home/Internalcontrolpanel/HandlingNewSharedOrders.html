Procedures to Follow When A New Shared Order is Arrived

 

New Order using Credit Card

  • First of all validate the order using Fraud detection process using steps mention in Fraud_detection page
    • Fraud Detection

 

  • If you find the order as fraud then send notice to client
    • Fraud Notice to Account  that is SUSPICIOUS
  • If a client don't reply within 4 hours, proceed with then contact Shabi/Ram/Mr Peter/Ms Minerva to get permission to void that order.
  • Make sure to do it within 5 hours because credit card transaction wont be voided after 5 hours and we will possibly received a charge back on it
  • Once you get permission to void order, click on "DELETE THIS ORDER" button. This will cancel order and void credit card transaction.
  • If by chance void window is passed or void is failed, clicking on "DELETE THIS ORDER" will refund transaction.
  • If a fraud already activated, then follow the cancellation procedure and contact Minerva to refund payment

New Order with PayPal

When customer place an order with paypal, then a payment notification email from paypal will receive to tickets.

The next step is verify order

  •  If order is legit, you can Activate account by clicking on "ACTIVATE THIS ORDER" button in order panel.

     
  • Match the paypal email address with order email address.
    • If its matching, then click on activate order.
    • If its not matching, then mail to paypal email address to authorize order. Do not activate order until they reply.
    • If there is no response after 7 days, then cancel order and contact Minerva to refund payment

New Orders with Check/Bank Payments

If payment type is check/money order or bank. Then first verify the order and then send the replies to customer contact email address for asking payment. 

If he reply then ask Minerva if that check/bank payment arrived.
