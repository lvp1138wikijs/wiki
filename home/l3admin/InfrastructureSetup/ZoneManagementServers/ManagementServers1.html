Ceph Monitor 1

Here we are discussing how to set up a ceph monitor server for the cluster. 

OS installed: CentOS 7. It is installed through our installation portal. 

Install the Packages

yum install epel-release -y
yum update -y 
yum install iptables-services epel-release lldpd python-setuptools -y
systemctl enable iptables lldpd

Add the Ceph Repository

cat >> /etc/yum.repos.d/ceph.repo <<END
[Ceph]
name=Ceph packages for $basearch
baseurl=http://download.ceph.com/rpm-luminous/el7/x86_64/$basearch
enabled=1
gpgcheck=1
type=rpm-md
gpgkey=https://download.ceph.com/keys/release.asc
priority=1
 
[Ceph-noarch]
name=Ceph noarch packages
baseurl=http://download.ceph.com/rpm-luminous/el7/noarch
enabled=1
gpgcheck=1
type=rpm-md
gpgkey=https://download.ceph.com/keys/release.asc
priority=1
 
[ceph-source]
name=Ceph source packages
baseurl=http://download.ceph.com/rpm-luminous/el7/SRPMS
enabled=1
gpgcheck=1
type=rpm-md
gpgkey=https://download.ceph.com/keys/release.asc
priority=1
END
