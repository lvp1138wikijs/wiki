Ceph Monitor 1

Here we are discussing how to set up a ceph monitor server for the cluster. 

OS installed: CentOS 7. It is installed through our installation portal. 

Install the Packages

yum install epel-release -y
yum update -y 
yum install iptables-services epel-release lldpd python-setuptools -y
systemctl enable iptables lldpd
