Bridge Server -2 & OOB

The management server-7 is the Secondary Bridge/Gateway server. We also install the KVM hypervisor in it and created a windows VM for console access. It's called the console server. The server has connections to the ipmi switch. That helps us to connect the servers via ipmi.

First, we have to set the server as a bridge. You can check this, Instruction to Setup Bridges

Then install the KVM hypervisor. Follow this page, How to Install KVM hypervisor.
