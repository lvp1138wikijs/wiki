Update NPD

For dedicated server account

 

Via ICP

Go to ICP » Search dedi/colo.

Then Search by hostname, serial number or username and click on “search”.

You will get the selected server and click on “View Server”.

You will see option right below “Accounting information” to update NPD . Put correct NPD and click on “Modify npd” to change

 

Via Insert

 

put_hostname_here        : Replace with the correct hostname.

update servertrack set npd = date_add(npd , interval 6 month) where hostname = "put_hostname_here";  (NOTE: This will add 6 months to the actual NPD)

OR

update userdata  set npd = "Date" where username = "UserName";

update sharedtrack set npd = "Date" where username = "UserName";

 

For shared accounts

put_username_here               : Replace with the correct Username

update sharedtrack set npd = date_add(npd , interval 36 month) where username = "put_username_here";  (NOTE: This will add 6 months to the actual NPD)

OR

update userdata  set npd = "2016-01-01" where username = "Put_username";

update sharedtrack set npd = "2016-01-01" where username = "Put_username";
