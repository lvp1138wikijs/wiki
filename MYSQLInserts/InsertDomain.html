Query to insert a domain so that it shows up in domain renewal screen. The last 4 parameters never change

Prepare the inserts in a single line and send it.

insert into registered_domains (domain,domainusername,domainregistered,firstname,lastname,orgname,address1,city,state,zip,country,phone,email,dayregistered,expdate,username,period,dns1,dns2,status,created) VALUES ('main_domain_name','domain_username_to be registered','domain_name_to be registered','FN','LN','Organisation_Name','Address1','City','State','Zip Code','Country','PH','Email','Day_registered','Exp_Date','Main_User_Name','1','dns1.serverpoint.com','dns2.serverpoint.com','A','1');

Example

normyip.com :: Main_domain_name

normyip :: Main_User_name

These informations are should be the account where you want the domain to show for renewal

mirkotur :: domain_username_to be registered

mirkoturla.com :: domain_name_to be registered
