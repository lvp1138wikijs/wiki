What the difference between QuickSSL, QuickSSL Premium, True BusinessID, and Wildcard SSL ?

QuickSSL and QuickSSL Premium are authenticated at the domain level only, whereas True BusinessID and Wildcard SSL are authenticated at the domain level and at the company/organization level.

  • Quick SSL is ideal for small to medium online businesses and shopping carts and online stores
  • Quick SSL Premium is ideal for medium to Large online Business and Business-2-Business (B2B)
  • True Business ID is ideal for large ecommerce retailers and High volume online businesses

SSL New Order Procedure

IP address & Domain

A dedicated IP address should be assigned to the domain name which is need SSL certificate.
SSL certificates are installed on fully qualified domain names (FQDN). That means if you purchase it for domain.com, it can not be installed on www.domain.com or subdomain.domain.com

1: Generate CSR

A customer signing request should be need to order SSL certificate

Key Size

CSR must have a minimum of 2048 bit key size.

 

  1. Genereate CSR Genereate CSR from cPanel
  2. Generate CSR from Plesk
  3. Generate CSR from IIS
  4. Generate CSR from Linux

If a client wants to Install SSL, then inform him Add payment to the account and get back us with the CSR, Domain name, Country, State, City, Organisation, and Email address that needs to send the SSL certificate.

Email Address: Validation email can only be sent to an email attached to the domain. like admin@domain.com or administrator / hostmaster / webmaster / or postmaster@domain.com. Those are the options and is for security reasons to prove they own the domain. Inform customer create and configure an email account with any of it before generating CSR

2: Order SSL

Once customer send CSR and the payment, then remove payment from customer using One time

Then contact Minerva with the following information to order SSL.

Client First/Last Name
Company Name
Address
City
State

Once SSL order, then authorization email and SSL cert will send to the email address he provided.

3: Install SSL

Customers can install it from their control panels. If they want us to install it, then tell them to provide an SSL certificate, RSA Private Key, and the CA bundle.

Renew Existing SSL Certificate

For Linux

  1. Check which SSL cert they already have and tell them to add a payment to their account.
  2. Once its added, then remove it using One time
  3. Then provided the domain information to Ms. Minerva to do a renewal request.
  4. Once it's done an approval email will send to the customer's email address. Inform customer to wait for the email and follow the instruction emails.
