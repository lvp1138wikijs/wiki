Nginx + PHP-FMP Installation Issue

REFERENCE TICKET ID LHO-388-37411

When we try to install Nginx with PHP-FPM from Plesk panel >> Tools and Settings > Updates and Upgrades. it finishes with the following message, but PHP-FPM was not installed

Installation started in background

Error: You already have the latest version of product(s) and all selected components installed. Installation will not continue.

When you check Nginx status, it will show following message and wont start.

root@:~# /etc/init.d/nginx status

* could not access PID file for nginx

root@:~# /etc/init.d/nginx start

* Not starting NGINX as it is disabled in config

To fix the issue, follow the steps given below:

apt-get install python-software-properties add-apt-repository ppa:yola/php5 apt-get update apt-get upgrade

/usr/local/psa/admin/bin/autoinstaller
/usr/local/psa/admin/bin/nginxmng --disable /usr/local/psa/admin/bin/nginxmng --enable /etc/init.d/nginx restart

/usr/local/psa/admin/sbin/httpdmng --reconfigure-all/etc/init.d/apache2 restart/etc/init.d/nginx restart
