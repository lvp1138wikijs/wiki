Create Second TEAMING on Windows Server 2012/2016

1. Open NIC teaming >> Select the current teaming "Team1 " under Teams

2. Go to Team Interfaces under Adapters and Interfaces

3.Then Click on Task >> Add Interface

4. Specify Vlan (For eg : vlan of simonfar is 22)

5. Save it.

This will create another interface with the NEW vlan.

6. You can see the new interface from Network and Sharing center.

7. Right click and go to its properties .

8. Add private IP and Subnet. Leave Gateway and DNS servers blank.

9. Save configuration.
