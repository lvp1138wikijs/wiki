The basic repository of Debian now contains OpenSSH version 4.x p which comes under portable and old versions of OpenSSH.

First we will check if new version is available in the source.

  1. apt-get update
  2. apt-get install openssh-server

If new version is not available we can install a more newer version of the OpenSSH from another source list. Need to add stable backports to the source list.

 

  1. Edit your /etc/apt/sources.list and add following line to it.
        deb http://ftp.debian.org/debian/ wheezy-backports main non-free contrib
  2. Now update the repo from the available list
        apt-get update

    Now install the OpenSSH from the new available package list

    apt-get -t wheezy-backports install openssh-server

You will now have a more newer version of the OpenSSH in your server.
