NcFTPd is a high-performance File Transfer Protocol (FTP) server software solution for UNIX systems, designed especially for high-traffic sites and internet service providers. NcFTPd is the FTP server written by NcFTP Software Inc http://www.ncftp.com/. The current version is: 2.8.7 .

Documentation Link : http://www.ice.mtu.edu/~jamyles/docs/ncftpd/doc/

/etc/shells must contain /bin/true and /bin/bash

Any username in /etc/ftpusers will be DENIED access via FTP. WARNING: sometimes after some Gentoo updates, Gentoo fills /etc/ftpusers with the usernames of ALL users in the system. Obviously, ncFTPd will deny access to all of them. Remove them and only leave system users, including root.
