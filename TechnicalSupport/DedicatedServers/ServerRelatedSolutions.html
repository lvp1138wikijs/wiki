PING

All data sent over the internet is sent in packets. Consider the following analogy. The idea behind packets is very similar to the idea of the capsules used to send checking and savings information from your vehicle to tellers inside the bank via vacuum tubes. The emails you send and the files you download are all broken down into raw data and inserted into little packets. These packets are piped through your Internet connection. When a packet arrives at a destination computer, the data is extracted and reassembled into a file.

Ping can be used to send packets of information to remote computers for the purpose of retrieving information. These programs are useful for testing your internet connection.

PING from windows

To run the ping command from windows system, go to Start → Run → type cmd and hit enter.

Then type the command either using the domain or IP

ping domain.com

ping IP

To stop press Ctrl+c

PING from Unix/Linux

To run the ping command from Unix/Linux system, the open terminal window then type the following command

ping domain.com or

ping IP

To stop press Ctrl+c

TELNET

Telnet is a user command and an underlying TCP/IP protocol for accessing remote computers.

Telnet from windows

To run the telnet command from windows system, go to Start » Run »type cmd and hit enter.

Then type the command either using the domain or IP

telnet domain.com port

telnet IP port

Telnet from Unix/Linux

It is a method of connecting to a remote computer over the Internet. To run the telnet command from Unix/Linux system, the Open terminal window then type the following command

telnet domain.com port or

telnet IP port

TRACEROUTE

Traceroute is a utility that records the route (the specific gateway computers at each hop) through the Internet between your computer and a specified destination computer. It also calculates and displays the amount of time each hop took. Traceroute is a handy tool both for understanding where problems are in the Internet network and for getting a detailed sense of the Internet itself.

TRACEROUTE from windows

To run the traceroute command from windows system, go to Start » Run »type cmd and hit enter.

Then type the command either using the domain or IP

tracert domain.com or

tracert IP

TRACEROUTE from Unix/Linux

To run the traceroute command from Unix/Linux system, the open terminal window then type the following command

traceroute domain.com or

traceroute IP

NSLOOKUP

The nslookup command can be used in Windows and Unix to find various details relating to DNS including IP addresses of a particular computer, MX records for a domain or the NS servers of a domain - all using DNS lookups. The name means “name server lookup”. The most common version of the program is included as part of the BIND package. A more modern alternative to nslookup is the dig program, also shipping with BIND.

NSLOOKUP from windows

To run the telnet command from windows system, go to Start → Run → type cmd and hit enter.

Then type:

nslookup domain.com

NSLOOKUP from Unix/Linux

To run the traceroute command from Unix/Linux system, the open terminal window then type the following command:

nslookup domain.com
