This feature only for Dedicated Linux Server

Turn ON Netboot

Then check server netbooted correctly . Put the server IP http://xxx.xxx.xxx.xxx in browser . It will show Server Maintenance Page.

Then browse http://xxx.xxx.xxx.xxx/serverpoint.php . Login with user root and password is the default password of OS images.

It will present you with a list of the partitions in that server

Under “all partitions”, select “check and correct”. It will started the File system check

When the fs check is finished, click on “reboot in normal mode” and that will reboot the machine and just turn off netboot . Don't reboot it again.

Note: Replace - xxx.xxx.xxx.xxx with correct server IP
