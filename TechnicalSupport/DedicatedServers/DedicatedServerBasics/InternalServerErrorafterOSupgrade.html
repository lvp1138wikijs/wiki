Issue: The forms present in customers server is giving 'Internal Server Error' after OS upgrade. The error in the logs is 'Premature end of script headers'. It is as given below:


----
[Thu Apr 10 14:30:57 2014] [error] [client 216.108.226.61] Premature end of script headers: FormMail.pl, referer: http://www.southtravels.com/middleeast/uae/kempinskihotelmall/enquiry.htm
----

Solution:
# mv /usr/sbin/suexec /usr/sbin/suexec.old
# cp /usr/local/psa/suexec/psa-suexec /usr/sbin/suexec
# chown root:apache /usr/sbin/suexec
# chmod 4510 /usr/sbin/suexec
# service httpd restart
