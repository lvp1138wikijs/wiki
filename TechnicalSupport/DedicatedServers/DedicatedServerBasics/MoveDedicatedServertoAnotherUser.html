This is procedure if customer want to change all his server to different user

Case 1: Move servers from one user another existing account

  • Make sure customer opened ticket from his authorized email address.
  • Ask customer his new account name, old account name and the servers they need to move to their new account.
  • Go to New ICP >> Edit client. Then search with old username.
  • Check is there any additional options they have activated. Like support plan, RAM, MsSQL, additional IPs etc. Make a note about the additional features.
  • Now prepare the following inserts and send to Ms Minerva
  • Once inserts made then verify accounts are moved correctly.
  • Now add those additional features to this new account.
  • Go to New ICP >> Edit client. Then search with new user and add those additional option to the exact servers.

 

Case 2: Move server from existing account to a new account/ownership

  • Make sure customer opened ticket from his authorized email address.
  • In this case customer need to signup a new colossus account from our website and link a credit card to that new account.
  • The we will get a new colossus order to the order system. Just activate the order.
  • Go to New ICP >> Edit client. Then search with old username and check is there any additional options they have activated. Like support plan, RAM, MsSQL, additional IPs etc. Make a note about the additional features.
  • Once account activated  then prepare the following inserts and send to Ms Minerva.
  • Once inserts made then verify accounts are moved correctly.
  • Now add those additional features to this new account. Go to New ICP >> Edit client. Then search with new user and add those additional option to the exact servers
