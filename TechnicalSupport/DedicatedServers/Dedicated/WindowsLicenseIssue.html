If a customer contacted us if his server license not activated. Then login to his server via Remote desktop. Then right click on My Computer and select properties. Below on that page you can see a message like this

 XX days left to activate the license

Product ID : XXXX-XXX-XXXXXX-XXXXXXX

Solution

The solution is install the correct key.

Go to Gmail account » Documents » Product Key » Find the exact key.

Login to the server » Right click on My computer » Go to Properties » Change the Product key.

Error message when you try to activate Windows : "Code 0x8007232b

Open File Product Keys : (File is Located in the Document Section of the Gmail Account)

Find the license number that correspond to the Windows version the server has installed

Go to Start –> Go to Computer and get its properties (left key). At the end of the screen you will open you will see the Windows Activation Area.
