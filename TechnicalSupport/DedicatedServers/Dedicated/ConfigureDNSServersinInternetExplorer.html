Issue: Browser can not load any website. 

 

Primary Resolver : 64.235.53.251

Secondary Resolver : 64.235.53.253

 

Go to control panel » Network Connections » Right click on the active connection and select Properties

Select the Internet Protocol option from the Drop down menu » Click on the Button Properties.

In the Properties window, select “Use the following DNS Server” option.

Then add the above IPs. This will fix the IE issue.
