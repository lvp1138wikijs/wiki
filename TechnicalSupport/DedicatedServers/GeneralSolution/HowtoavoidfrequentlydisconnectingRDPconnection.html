In order to avoid frequently disconnecting RDP connection, please do the following steps: ( Reference ticket : XAE-268-52225 )

  1. Remove/uninstall "sound over RDP" application from VM.
  2. In RDP client setting window, click "options" and then select the "Local Resources" tab.
  3. In "Remote audio" settings, please make sure that "Remote audio playback" and "Remote audio recording" set to Do not play/Do not record respectively.
