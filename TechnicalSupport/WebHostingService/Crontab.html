To setup the cron job customer will have to login to his account via SSH.

Place the script file at root of the account, That is outside public_html and then Then just need to enter that script in to cronjob so it will be executed automatically at specified time.

To add a cronjob use the command "crontab -e", An editor will be opened,

-Press insert to go in insert mode

- Then he can write the cront job text in it in below format.

MINUTE HOUR DAY MONTH DAYOFTHEWEEK COMMAND

  

Example:

0 * * * * test.pl

OR

0 * * * * php test.php

This will execute each hour (*:00), every day

 

59 23 30 4,6,9,11 * /home/kira/cmd2.pl

This will execute the program at 23:59pm, on the 30th day of the month, on the 4, 6, 9 and 11 of each month.

After the editing, Press escape and then :wq to save and exit.

we are using litespeed webserver therefore php in your account is lsphp rather then php. For litespeed change php to lsphp in crontab.
