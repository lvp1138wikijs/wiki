Step1: Go to the directory you want to setup password protection and run the following code

Replace username with correct account username

htpasswd -c .htpasswd username
htpasswd -c .htaccess username

Step2: Create a  .htaccess file. Use your favorite editor like vi, emac. 

sample .htaccess file is like this, replace domainname with account domain name

AuthType Basic
AuthName domainname
AuthUserfile ./.htpasswd

<Limit GET>
require valid-user
</Limit>

Step3: Change permissions

chmod 755 .htaccess
chmod 755 .htpasswd
