If customer need Zend Framework, Which is not something installed on server level. It is a part of application and if it is not included in their application. We just need to download Zend Framework and put it in his public_html directory under Zend directory

Here the steps for installing zend framework

Login to the account shell prompt via ssh

Download the Zend Framework and extract it

wget http://framework.zend.com/releases/ZendFramework-1.11.4/ZendFramework-1.11.4-minimal.tar.gz 
tar -zxf ZendFramework-1.11.4-minimal.tar.gz

Locate the Loader.php script

find ./ -name Loader.php

The result should like this ./ZendFramework-1.11.4-minimal/library/Zend/Loader.php

Copy the folder Zend to public_html

cp -rf ./ZendFramework-1.11.4-minimal/library/Zend  ./public_html/

Now the Zend location is ./public_html/Zend
