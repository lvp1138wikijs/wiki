Default login credentials

Username is “sysadmin” and password is “PASS”.

Reset its password

Connect to the KVM device via a serial connection. Press ENTER several times until you see the console prompt.

Enter “reset” and answer “yes”. Immediately after that, press ESC several times. It will now return you back to a prompt. There, enter “defaults”. It should then acknowledge the request with “ok”.

At that point, reboot, and start configuration again.

This is what you should see:

===== Welcome! Choose a command for the following features: -Initial IP configuration: "config". -Reset device: "reset". [x.x.x.x spider]> reset To reset the device enter "yes": yes 0 <-- This is when you press Esc a few times => defaults Please wait... ok =====
