Blocked Ports, IRC

If a customer complains that some ports on their dedicated server are blocked.

For example.

I have two non-linked irc servers however the portst 6667 etc (irc) are blocked. I do not see anything at your AUP about this block.

 

In such cases you can provide the following details to the customer.

  • We do not block any ports on our network.
  • If you are not able to access these ports it might be blocked on the default firewall comes with the server.

 

For Example:

CentOS has a default firewall enabled by default.

To see the rules on the firewall, Execute the following command.

iptables --list -nv

To remove those rules, clear out the following file

/etc/sysconfig/iptables

and execute the following commands:

iptables -F

iptables -X
