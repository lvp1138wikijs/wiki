How to share your ssh terminal screen. So that we do not need to provide the person with server root credentials and monitor what all commands and updates are done from our terminal.

Step-by-step guide

 

ssh terminal sharing with "screen" and it is much simple. These are the steps to be followed:

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Step1: create a test user
useradd -m USERNAME

Step2: Set password
passwd USERNAME

sTEP 3: Switch to that user: 
su - USERNAME

sTEP 4: start a session
screen -S session name

Now ask that person to connect to that server providing the test user name and password:
Then provide the command to attach to the screen using the following command:

screen -ls
screen -x "screen-name"

That's it. we can then monitor and enter the passwords from our screen.
